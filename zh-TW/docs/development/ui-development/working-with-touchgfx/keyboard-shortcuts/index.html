TouchGFX Simulator Features​

Simulator​

Enables/disables debug info
Enables/disables highlighting invalidated area
Takes a screenshot and places the image under the screenshots folder
Takes screenshots of the next 50 frames and places the images under the screenshots folder
Takes a screenshot and places it in your clipboard
If a simulator skin is used - enables/disables the simulator skin
