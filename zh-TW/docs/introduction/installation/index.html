安裝

TouchGFX以X-CUBE-TOUCHGFX壓縮檔案的形式提供，其中包含以下元件：

  • TouchGFX Designer - 基於Windows的GUI建構工具來建構UI
  • TouchGFX Generator - 通過CubeMX創建自定義TouchGFX HAL層
  • TouchGFX Engine - 驅動UI應用程式的TouchGFXC++框架

在STM32評估套件上進行原型設計​

如果您只是想試用TouchGFXDesigner並在STM32評估套件上進行一些原型設計，請參考安裝TouchGFX Designer章節。

安裝TouchGFX Designer​

從ST.com官網將X-CUBE-TOUCHGFX下載到硬碟的任何位置，並解壓縮。

從st.com上下載X-CUBE-TOUCHGFX

在解壓縮的資料夾內，您可以在以下路徑中找到TouchGFX.msi安裝程式：

Utilities\PC_Software\TouchGFXDesigner

按兩下.msi檔案可打開安裝程式。 按照說明完成安裝。

安裝TouchGFX Designer

Tip
您需要使用以下工具，透過TouchGFXDesigner來燒錄您的板子：
  • STM32CubeProgrammer
  • STM32ST-LINKUtility

安裝STM32CubeProgrammer​

After downloading STM32CubeProgrammer from STM32CubeProgrammer download location, uncompress the downloaded .zip file and launch the .exe installer file. 然後按照說明完成安裝。

安裝TouchGFX Designer

Caution
為了讓TouchGFXDesigner與Makefile使用STM32CubeProgrammer來燒錄開發板，必須將其安裝在預設的安裝位置： C:\ProgramFiles\STMicroelectronics\STM32Cube\STM32CubeProgrammer

安裝STM32ST-LINKUtility​

After downloading STM32 ST-LINK Utility from STM32 ST-LINK utility location, simply open the downloaded .exe file and follow the instructions to complete the installation process.

安裝STM32ST-LINKUtility

Caution
為了使TouchGFXDesigner和Makefile能夠使用STM32ST-LINKUtility來燒錄目標硬體，必須將其安裝在預設安裝位置： C:\Program Files (x86)\STMicroelectronics\STM32 ST-LINK Utility

客製化產品開發​

如果您想在意法半導體的顯示器開發套件或基於STM32的自定義平臺上執行TouchGFX應用程式，請參考在CubeMX中安裝TouchGFXGenerator章節。

在CubeMX中安裝TouchGFX Generator​

在CubeMX中安裝TouchGFXGeneratorCubeMX安裝X-CUBE-TOUCHGFX，請先選擇"求助"下的"管理嵌入式套件"（或按)

說明-%gt;管理嵌入式套件

按下「刷新」以獲取可更新的套件清單。

刷新以更新可用的軟體包

跳到STMicroelectronics選項卡。 找到X-CUBE-TOUCHGFX"，並展開。 選取「TouchGFXGenerator」，然後點擊「立即安裝」。 將下載軟體包並顯示授權協議。

找到TouchGFXGenerator套件

閱讀並接受授權協議，然後點擊「完成」，以在CubeMX中安裝TouchGFXGenerator。

接受授權協議

X-CUBE-TOUCHGFX現在被解壓到以下路徑中：

C:\Users\<user>\STM32Cube\Repository\Packs\STMicroelectronics\X-CUBE-TOUCHGFX\4.13.0
