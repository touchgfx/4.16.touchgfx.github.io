インストール

TouchGFXはX-CUBE-TOUCHGFX zipファイルとして配布され、以下のコンポーネントを含みます。

  • TouchGFX Designer：WindowsベースのGUIビルダを通してUIを構築
  • TouchGFX Generator ：UIアプリケーションを動作させるTouchGFX C++フレームワーク
  • TouchGFX エンジン：UIアプリケーションを動作させるTouchGFX C++フレームワーク

STM32評価キットでのプロトタイプ作成​

TouchGFX Designerを使用して、STM32評価キットで何らかのプロトタイプを作成してみたいとをお考えの場合は、「TouchGFX Designerのインストール」セクションをご参照ください。

TouchGFX Designerのインストール​

公式Webサイトのst.comから、X-CUBE-TOUCHGFXをお使いのハード･ドライブ上の任意の場所にダウンロードし、抽出します。

st.comからX-CUBE-TOUCHGFXをダウンロード

抽出したフォルダ内の次のパスにTouchGFX .msiインストーラがあります。

Utilities\PC_Software\TouchGFXDesigner

.msiファイルをダブルクリックすると、インストーラが起動します。 指示に従ってインストール･プロセスを完了します。

TouchGFX Designerのインストール

Tip
TouchGFX Designerからボードに直接Flashできるようにするには、以下のツールが必要です
  • STM32CubeProgrammer
  • STM32 ST-LINKユーティリティ

STM32CubeProgrammerのインストール​

STM32CubeProgrammerのダウンロード場所 からSTM32CubeProgrammerをダウンロードした後、ダウンロードされた .zip ファイルを解凍して、 .exe インストーラ･ファイルを起動します。 その後、指示に従ってインストール･プロセスを完了します。

TouchGFX Designerのインストール

Caution
ターゲット･ハードウェアにFlashするために、TouchGFX DesignerとMakefileでSTM32CubeProgrammerを使用できるようにするには、これを次のデフォルトのインストール場所にインストールする必要があります。 C:\Program Files\STMicroelectronics\STM32Cube\STM32CubeProgrammer

STM32 ST-LINKユーティリティのインストール​

STM32 ST-LINKユーティリティの場所 からSTM32 ST-LINKユーティリティをダウンロードした後、ダウンロードされた .exe ファイルを開き、指示に従ってインストール･プロセスを完了します。

STM32 ST-LINKユーティリティのインストール

Caution
ターゲット･ハードウェアにFlashするために、TouchGFX DesignerとMakefileでSTM32 ST-LINKユーティリティを使用できるようにするには、これを次のデフォルトのインストール場所にインストールする必要があります。 C:\Program Files (x86)\STMicroelectronics\STM32 ST-LINK Utility

カスタム製品開発​

STディスプレイ･キットまたは独自カスタムのSTM32ベース･プラットフォームのいずれかでTouchGFXアプリケーションを実行する場合は、「CubeMXでのTouchGFX Generatorのインストール」セクションを参照してください。

CubeMXでのTouchGFX Generatorのインストール​

CubeMXを介してX-CUBE-TOUCHGFXをインストールするには、まず、［Help］の下の［Manage Embedded Software Packages］を選択します（またはAlt + UAlt + Uを押します）。

［Help］ -%gt; ［Manage embedded software packages］

［Refresh］をクリックして、使用可能なパッケージの最新のリストを表示します。

［Refresh］をクリックして使用可能なパッケージを更新

［STMicroelectronics］タブに移動します。 スクロールして［X-CUBE-TOUCHGFX］を見つけ、このノードを展開します。 [TouchGFX Generator］チェックボックスをチェックして、［Install Now］をクリックします。 パッケージがダウンロードされ、ライセンス契約が表示されます。

TouchGFX Generatorパッケージを見つける

ライセンス契約を読んで同意し、［Finish］をクリックして、CubeMX内にTouchGFX Generatorをインストールします。

ライセンス契約に同意

X-CUBE-TOUCHGFXディストリビューションが次のパスにアンパックされます。: 

C:\Users\<user>\STM32Cube\Repository\Packs\STMicroelectronics\X-CUBE-TOUCHGFX\4.13.0
